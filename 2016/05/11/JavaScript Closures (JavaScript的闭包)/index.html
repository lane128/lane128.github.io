JavaScript Closures (JavaScript的闭包)

JavaScript Closures

先看个例子：

这里变量 add 被分配给一个自引用方法的返回值。

​ =>这里我们可以理解 add = function() {return counter += 1;}

这个匿名函数只运行一次，它把变量counter 设置成0，并且返回一个function表达式。

通过这种方法 add 变成了一个方法，更神奇的地方是它能访问父方法的定义域。

这就叫做javascript的闭包。这种机制让javascript的方法能拥有“私有”变量。

counter 被匿名函数的定义域保护，只能被add方法改变。
